# Stock BOS Detector

A sophisticated trading tool that detects BOS (Break of Structure) and CHOCH (Change of Character) patterns across all 100 Nifty 100 Index stocks.

## Features
- Real-time pattern detection
- Multiple timeframes (Daily/Weekly/Monthly) 
- Advanced technical analysis
- Professional UI with filtering
- **100 Nifty 100 Index stock coverage** (Nifty 50 + Nifty Next 50)
- 📥 **Export to Excel** - Download results with all details
