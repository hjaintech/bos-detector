# Stock BOS Detector

A sophisticated trading tool that detects BOS (Break of Structure) and CHOCH (Change of Character) patterns across 50 Nifty Index stocks.

## Features
- Real-time pattern detection
- Multiple timeframes (Daily/Weekly/Monthly) 
- Advanced technical analysis
- Professional UI with filtering
- 50 Nifty Index stock coverage
